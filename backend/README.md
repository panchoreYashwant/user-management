## Prerequisites

- Node.js
- npm or yarn

## Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/panchoreYashwant/user-management.git
   cd IPANGRAM-APP
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

1. Start the server:

   ```bash
   npm run start
   ```

2. Access at `http://localhost:5000`.

## API Endpoints

- **GET** `/api/stocks`: Endpoint description
